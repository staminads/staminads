# v3.3.0

## Privacy Settings
- Add Privacy settings tab in workspace settings
- Geo privacy controls: enable/disable geo tracking, city/region storage, coordinates precision (0-2 decimal places)

## Default Traffic Source Filters
New workspaces are now pre-populated with 39 default filters for automatic channel attribution:

**Paid channels (via click ID detection):**
- Google Ads (gclid/gbraid/wbraid), Facebook Ads (fbclid), Microsoft Ads (msclkid)
- TikTok Ads (ttclid), Pinterest Ads (epik), LinkedIn Ads (li_fat_id), Twitter Ads (twclid)
- Snapchat Ads (ScCid), Reddit Ads (rdt_cid), Quora Ads (qclid)

**Paid channels (via UTM):**
- Google, Microsoft/Bing, Facebook, Instagram, LinkedIn, TikTok, YouTube

**Organic search:**
- Google, Bing, Yahoo, DuckDuckGo, Baidu, Yandex (+ Android Google Search app)

**Social organic:**
- Facebook, Instagram, Twitter/X (including t.co), LinkedIn, YouTube, TikTok, Pinterest, Reddit, Snapchat, Quora

**Other:**
- Direct traffic, Email marketing, Google Ads Display (googleadservices.com)
- Default fallback (not-mapped)
